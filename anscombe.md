# anscombe — 앤스컴 사중주 데이터

> F.J. Anscombe(1973)이 시각화의 중요성을 보이기 위해 만든 4개 데이터셋. 통계 수치는 같지만 분포가 전혀 다름.


---


## 기본 정보

- **행 수**: 44개
- **열 수**: 3개
- **주요 용도**: 시각화 필요성 강조, 선형 회귀 잔차 분석, 통계 vs 시각화 비교
- **시험 활용**: ADP (통계 이론, 시각화 중요성)

## 컬럼 설명

| 컬럼명 | 타입 | 설명 |
|---|---|---|
| `dataset` | object | 데이터셋 ID (I / II / III / IV) |
| `x` | float64 | 독립 변수 |
| `y` | float64 | 종속 변수 |

## 샘플 데이터 (상위 3행)

| dataset | x | y |
|---|---|---|
| I | 10.0 | 8.04 |
| I | 8.0 | 6.95 |
| II | 10.0 | 9.14 |
| III | 10.0 | 7.46 |
| IV | 8.0 | 6.58 |

## 코드 예시

```python
import seaborn as sns

df = sns.load_dataset('anscombe')

# 4개 그룹 통계 비교 (거의 동일!)
print(df.groupby('dataset')[['x','y']].agg(['mean','std','corr']))

# 실제로는 분포가 전혀 다름 → 시각화 필수
```
